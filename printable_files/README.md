# Printable files

Ready-to-slice **STL files for the whole collection**, named
`roller_<name>_<theme>.stl` (plus Ember's `.3mf`). Each is a single watertight,
upright, support-free solid with a top-end press-stamp.

Regenerate or add more with `playdoh_roller.py --stl`. Note these are large
(~60 MB each); Git LFS is an option if the repo gets unwieldy.
