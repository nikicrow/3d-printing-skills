# Printable files

A few **example STLs are kept here** (Ember / Imogen / Mikey, plus Ember's
`.3mf`) so the repo has ready-to-slice samples.

Additional rollers are generated by `playdoh_roller.py --stl`. New `*.stl` /
`*.3mf` are **git-ignored** by default (large binaries) so the repo stays light —
regenerate any roller on demand. The committed example files above are kept
explicitly (tracked despite the ignore rule).
