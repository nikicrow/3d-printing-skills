# Decoration icon attribution

All decoration SVGs were fetched from open icon sets via the Iconify API
(https://iconify.design) and rasterized into the roller pattern. They are used
as bold silhouette stamps.

| File | Source icon | Set | License |
|---|---|---|---|
| bee.svg | fluent-emoji-high-contrast:honeybee | [Microsoft Fluent Emoji](https://github.com/microsoft/fluentui-emoji) | MIT |
| flower.svg | mdi:flower | [Material Design Icons](https://pictogrammers.com/library/mdi/) | Apache 2.0 |
| trex.svg | noto:t-rex | [Google Noto Emoji](https://github.com/googlefonts/noto-emoji) | Apache 2.0 / CC BY 4.0 |
| brontosaurus.svg | noto:sauropod | Google Noto Emoji | Apache 2.0 / CC BY 4.0 |
| cat.svg | mdi:cat (cat head/face) | [Material Design Icons](https://pictogrammers.com/library/mdi/) | Apache 2.0 |
| paw.svg | teenyicons:paw-solid | [Teenyicons](https://github.com/teenyicons/teenyicons) | MIT |
| banana.svg | noto:banana | Google Noto Emoji | Apache 2.0 / CC BY 4.0 |
| apple.svg | noto:red-apple | Google Noto Emoji | Apache 2.0 / CC BY 4.0 |
| circle.svg | mdi:circle | [Material Design Icons](https://pictogrammers.com/library/mdi/) | Apache 2.0 |
| square.svg | mdi:square-rounded | Material Design Icons | Apache 2.0 |
| triangle.svg | mdi:triangle | Material Design Icons | Apache 2.0 |
| star.svg | mdi:star | Material Design Icons | Apache 2.0 |
| heart.svg | mdi:heart | Material Design Icons | Apache 2.0 |

Multicolor Noto emoji are flattened to a single silhouette (union of all
sub-paths) at rasterization time. All other icons are single-colour and use
even-odd fill so interior holes (flower centre, bee wings, cat eyes) are kept.
